# library-management-system-OPP
A desktop database management application engineered utilizing **Java Standard Edition (Java SE)** and core **Object-Oriented Programming (OOP)** methodologies. This system models real-world library administrative tasks, automated book tracking workflows, member transaction logs, and search functionalities under an integrated architecture.

---

## 🚀 Core OOP Design Patterns Applied
* **Encapsulation:** Protects entity attributes (Books, Users) using strict private modifiers and access control endpoints.
* **Inheritance & Polymorphism:** Extends base customer schemas to dynamic administrative roles with distinct behavioral executions.
* **Abstraction:** Implements functional interface definitions to decouple infrastructure layers from backend data operations.
